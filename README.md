## CLI Flags Reference

| Flag                  | Example                      | What it Does                     | Why It’s Useful                | Status         |
| --------------------- | ---------------------------- | -------------------------------- | ------------------------------ | -------------- |
| `--depth`, `-d`       | `--depth 3`                  | Limits call hierarchy depth      | Prevents huge output           | ✅ Implemented |
| `--roots-only`        | `--roots-only`               | Shows only entry-point functions | Identifies app starting points | ✅ Implemented |
| `--json`              | `--json`                     | Outputs call tree as JSON        | Machine-readable output        | ✅ Implemented |
| `--focus <fn>`        | `--focus initApp`            | Shows tree only from a function  | Targeted analysis              | ✅ Implemented |
| `--lang <lang>`       | `--lang js`                  | Forces language parser           | Needed for multi-lang support  | ❌ Not yet     |
| `--recursive`, `-r`   | `-r ./src`                   | Scan directories recursively     | Real projects use folders      | ❌ Not yet     |
| `--exclude-dir`       | `--exclude-dir node_modules` | Skip directories                 | Avoid noise                    | ❌ Not yet     |
| `--include-anonymous` | `--include-anonymous`        | Include anonymous functions      | Optional deeper analysis       | ❌ Not yet     |
| `--include-arrow`     | `--include-arrow`            | Include arrow functions          | JS-heavy codebases             | ❌ Not yet     |
| `--include-methods`   | `--include-methods`          | Include class methods            | OOP support                    | ❌ Not yet     |
| `--exclude-builtins`  | `--exclude-builtins=false`   | Ignore built-in functions        | Cleaner output                 | ❌ Not yet     |
| `--format`            | `--format tree`              | Output style (tree/json/dot)     | Flexible exports               | ❌ Not yet     |
| `--sort`              | `--sort alpha`               | Sort nodes alphabetically        | Stable output                  | ❌ Not yet     |
| `--line-numbers`      | `--line-numbers`             | Show function start/end lines    | Navigation aid                 | ❌ Not yet     |
| `--show-file`         | `--show-file`                | Show file per function           | Multi-file support             | ❌ Not yet     |
| `--detect-cycles`     | `--detect-cycles`            | Detect recursive loops           | Safety & clarity               | ❌ Not yet     |
| `--collapse-cycles`   | `--collapse-cycles`          | Collapse recursion visually      | Prevent infinite trees         | ❌ Not yet     |
| `--max-nodes`         | `--max-nodes 100`            | Stops output after N nodes       | Performance control            | ❌ Not yet     |
| `--stdin`             | `--stdin`                    | Read source from STDIN           | Piping support                 | ❌ Not yet     |
| `--config`            | `--config calltree.yaml`     | Load config defaults             | Team usage                     | ❌ Not yet     |
| `--debug`             | `--debug`                    | Verbose internal logs            | Dev troubleshooting            | ❌ Not yet     |
| `--dump-ast`          | `--dump-ast`                 | Print Tree-sitter AST            | Grammar debugging              | ❌ Not yet     |
| `--dump-functions`    | `--dump-functions`           | Print extracted functions        | Parser validation              | ❌ Not yet     |
| `--dump-calls`        | `--dump-calls`               | Print raw call edges             | Graph debugging                | ❌ Not yet     |
| `--watch`             | `--watch`                    | Re-run on file change            | Dev convenience                | ❌ Not yet     |
| `--version`           | `--version`                  | Show tool version                | Standard CLI behavior          | ❌ Not yet     |
| `--help`              | `--help`                     | CLI help output                  | Auto-generated by Cobra        | ✅ Built-in    |
